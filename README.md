# React App Shopping Cart

This is a simple shopping cart where products are displayed in a list and a free shipping bar is shown when total price is more than R$10.

## Installation

```bash
cd server
node app.js

../

cd client
npm start
```

## Stack

React, Typescript, materialUI.
